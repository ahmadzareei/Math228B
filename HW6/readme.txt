All files are compile with  g++ (Debian 4.8.2-16) 4.8.2
files ending with ".cpp" are compiled in terminal using
>> g++ filename.cpp -o out
and runned wih
>>./out


files ending in ".gnup" are used for plotting. All plots are build using gnuplot 4.6 (Debian)
running these files, in terminal one can write 
>>gnuplot filename.gnup
